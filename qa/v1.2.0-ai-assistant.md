# QA Funcional · v1.2.0 AI Assistant

## Objetivo
Validar el Asistente IA basado en reglas, sin API externa.

## Casos
- Abrir Asistente IA en sistema real.
- Ver KPIs: insights activos, prioridad alta, cobranza e inventario.
- Validar lista de prioridades con enlaces a módulos.
- Validar pacientes con contexto inteligente.
- Abrir Ficha 360° y revisar resumen IA/siguiente acción.
- Validar Dashboard con métrica Insights IA.
- Abrir demo del Asistente IA y confirmar mismo comportamiento con datos mock.
- Ejecutar `sql/reset_full_database.sql` en entorno de prueba y confirmar que:
  - elimina pacientes/citas/tratamientos/presupuestos/pagos,
  - conserva configuración,
  - precarga servicios base,
  - precarga insumos base,
  - no afecta demo.

## Resultado esperado
El módulo orienta acciones sin depender de OpenAI ni WhatsApp real.
